# 產品型錄 PWA - 使用說明

## Google Sheets 設定步驟

### 1. 建立 Google Sheets
在 Google Sheets 建立新試算表，欄位如下：

| A 欄 | B 欄 | C 欄 | D 欄 | E 欄（選填）|
|------|------|------|------|------------|
| 產品名稱 | 分類 | 圖片連結 | 產品描述 | 排序 |

- **分類**：女裝、童裝、生活用品（可自由新增）
- **圖片連結**：Google Drive 分享連結 或 圖床連結
- **排序**：數字越小排越前面，不填則預設 999

### 2. 發布試算表
1. Google Sheets → 「檔案」→「分享」→「發布到網路」
2. 選擇「整份文件」→ 格式選「CSV」
3. 點「發布」→ 複製產生的連結

### 3. 設定連結
將複製的連結貼到 `app.js` 最上方的 `SHEET_CSV_URL` 變數中。

### 4. 圖片上傳建議
- 建議尺寸：800 x 800 像素（正方形）
- 如用 Google Drive：上傳圖片 → 右鍵「分享」→「取得連結」→ 改為「知道連結的人都能檢視」
- Google Drive 圖片連結格式：`https://drive.google.com/uc?id=圖片ID`

## 部署
可部署至 GitHub Pages、Netlify 或 Vercel（皆免費）。
